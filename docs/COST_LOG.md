# Cost Log

This project follows a **zero-budget** rule: no paid APIs, no paid data,
no cloud-warehouse scans, no private or institutionally restricted data.

| Item | Cost | Notes |
|---|---|---|
| Stack Exchange Data Explorer (SEDE) | $0 | Free public browser interface |
| Stack Exchange API v2.3 | $0 | Free public endpoint; a registered key raises the quota from 300 to 10,000 requests/day at no charge |
| Sentence-embedding models (`all-MiniLM-L6-v2`, `all-mpnet-base-v2`) | $0 | Free pre-trained models run locally |
| Compute | $0 | Local machine only |
| Storage | $0 | Local disk |

No credit card, paid API quota, BigQuery scan, or commercial dataset was
used at any point. The 2025 extension was collected with a free
registered application key. Where a data source requires registration
but no payment (e.g., the API key, IPUMS CPS), that is documented and no
paywall is bypassed.

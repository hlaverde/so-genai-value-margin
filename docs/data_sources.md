# Data Sources

This project uses only zero-budget public data. All source access conditions should be re-checked before publication because terms may change.

## Stack Exchange Data Explorer / Stack Overflow

- Official URL: https://data.stackexchange.com/help
- Role in project: core data source for tag-week, user-tag-week, and post-level complexity panels.
- Access mode: public Stack Exchange Data Explorer queries, manually downloaded as CSV.
- Main tables: `Posts`, `Users`, `Votes`, `PostTags`, `Tags`, and `Comments`.
- Cost/access condition: free public web interface. Query results are manually exported, avoiding paid APIs.
- License/use notes: SEDE describes itself as a tool for querying public Stack Exchange network data. Stack Exchange public user contributions are under Creative Commons Attribution-ShareAlike licenses, with version depending on contribution date. The repository must preserve citation and attribution notes when publishing derived outputs.

## GH Archive

- Official URL: https://www.gharchive.org/
- Role in project: extension data for GitHub event panels by language-week, repository-week, and user-week.
- Access mode: selective HTTP download of hourly `.json.gz` event files.
- Events of interest: `PushEvent`, `PullRequestEvent`, `IssuesEvent`, `IssueCommentEvent`, `ForkEvent`, `CreateEvent`, and `WatchEvent`.
- Cost/access condition: free public HTTP archives. Scripts should download only narrow windows needed for pre-specified panels.
- License/use notes: GH Archive archives public GitHub event data. Use must respect GitHub event visibility and GitHub terms. No private repository data are used.

## O*NET Database

- Official URL: https://www.onetcenter.org/database.html
- Role in project: secondary occupational module for tasks, work activities, skills, technologies, experience, and occupation mappings.
- Access mode: public downloadable database files.
- Cost/access condition: free public download.
- License/use notes: O*NET Resource Center states that the majority of O*NET information is available under Creative Commons Attribution 4.0 International. Attribution is required.

## IPUMS CPS

- Official URL: https://cps.ipums.org/cps/
- Role in project: secondary labor-market validation, not the core empirical design.
- Access mode: free IPUMS CPS extract system after user registration and terms acceptance.
- Cost/access condition: free online extraction system, but registration and terms of use are required.
- License/use notes: IPUMS requires appropriate citation and asks users to provide publication citations for work using IPUMS data. No restricted-use data or institutional agreements are allowed in this project.

## BLS OEWS / QCEW

- OEWS official URL: https://www.bls.gov/oes/tables.htm
- QCEW official URL: https://www.bls.gov/cew/downloadable-data-files.htm
- Role in project: secondary context on occupational employment, wages, industries, and geography.
- Access mode: public BLS downloadable tables/files.
- Cost/access condition: free public files.
- License/use notes: BLS is a U.S. federal statistical agency; BLS data products are generally public government data. Citation to BLS tables, release dates, and downloaded files should be included.

## Data-Governance Rules

- Raw data are never edited manually.
- Raw downloads are stored under `data/raw/` or `data/external/`, which are ignored by git.
- Intermediate and processed files are generated by scripts.
- Every query, filter, and transformation must be documented in code or in `docs/replication_notes.md`.
- BigQuery is not part of the baseline workflow. Any future BigQuery script must estimate bytes scanned before execution and abort if the estimate exceeds a free-quota threshold.

# Overleaf Project

Upload the contents of this folder, or upload the generated ZIP file, to Overleaf.

Main file:

- `main.tex`

Included folders:

- `sections/`: article text.
- `tables/`: generated LaTeX tables.
- `figures/`: generated PNG figures.

The empirical tables and figures are generated from the reproducible Python pipeline in the repository. Do not edit generated tables by hand; regenerate them from scripts when results change.
